MRP Timesheet - Manufacturing Timesheet Odoo Apps

Manufacturing Workorder Timesheet Odoo Apps

MRP Timesheet - Manufacturing Timesheet Odoo App helps users to manage timesheets directly from manufacturing order and work order by using 'Manage Timesheet' checkbox and also users can use the timesheet smart button to easily view timesheet records. When users fill a timesheet in work order that records will reflect in manufacturing orders timesheet.

Features

Manage Timesheet Directly from Manufacturing Order

User can tick on 'Manage Timesheet' checkbox for easily manage timesheet on manufacturing order.

Manage Timesheet Directly from Work Orders

User can easily manage timesheet directly from work orders and that records will reflect in manufacturing orders timesheet also.
Demo Request Video Tutorial

Manufacturing Order

User can see 'Manage Timesheet' checkbox in manufacturing orders form view.

Manage Timesheet for Manufacturing Order

When user tick on 'Manage Timesheet' checkbox, it will display 'Timesheet' tab in manufacturing orders and user can easily manage timesheet.

Manage Timesheet for Work Orders

User can easily manage timesheet for work orders and if user fill timesheet in work orders, It will reflect in manufaturing orders timesheet also.

My Timesheets in List View

User can easily view his/her own timesheets in list view.
